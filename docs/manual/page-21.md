ii. A value that is higher or equal to the FF position will cause the song to stop.
### 48. Disable all MIDI

a. Set the MIDI Port to 9999 within the .cfg file to disable all MIDI processing

i. No MIDI calls or initialization will take place when this is enabled

### 50. Export to .WAV
