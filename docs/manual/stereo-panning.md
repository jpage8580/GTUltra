### 7. Stereo Panning

a. When playing in either mono or stereo mode (not True Stereo) it is possible to set the panning position of each SID chip.
b. By default, SIDs 1+3 will play from the left speaker, and SIDs 2+4 will play from the right speaker. However, in many cases (for example, when playing 3
    SID music), it may be that you want SIDs laid out in as Left, Right, Center.
c. By Shift or Ctrl clicking on the HR: text at the top of the editor, the view will change to the panning information for the number of SID chips that is
    currently selected.
d. For example, if 3 SID chips are selected:

    ![image](../images/manual/stereo-panning-img-1.png)

    i. Here, you can see that P3 (Panning for 3 SID chips) information is showing 3 panning values (hard-left, hard-right and center)
e. If 4 SID chips are selected for playback, the view shows 4 values:

    ![image](../images/manual/stereo-panning-img-2.png)

    i. As you can see, panning is set to Left,Right,Left,Right for each SID chip
f. This panning information (a set of 4 values for SIDs 1-4) is saved in the GTUltra.cfg file
