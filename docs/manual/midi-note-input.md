### 14. MIDI note input

a. Disable all MIDI processing by setting the MIDI Port number to 9999 within the GTUltra.cfg file or via -m command line option
b. Connecting a MIDI keyboard  BEFORE STARTING GT-ULTRA  allows for MIDI keyboard input.

    i. MIDI keyboard input works in jam mode when also editing tables / instruments
c. Shift-click on the keyboard button in the transport bar to go to Midi Settings panel
