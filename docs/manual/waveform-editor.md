### 36. Waveform editor

![image](../images/manual/waveform-editor-img-1.png)

a. When editing any waveform parameter, the current waveform settings are displayed (replacing the timer display above the transport bar)
b. Click on the waveforms / test bit (TEST), ringmod (RING), sync or gate bit (GATE) information to toggle enable / disable
