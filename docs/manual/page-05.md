## Features compared to GTStereo

### 1. Updated display / skinning

a. User definable colour schemes

i. Make it easier to visualise what is actually going on!
ii. Presets are aimed to make you feel comfortable, with skins based on other trackers. If you don’t like them - create your own!
### 2. Undo (ctrl-z)

a. Yes. Really.
### 3. Instrument Use count (IC)

a. Displays how many times an instrument has been used in all patterns
b. Allows you to quickly find unused instruments
